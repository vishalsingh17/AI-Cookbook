# Python
Python is general purpose, high-level language which is used widely in domains like Artificial Intelligence, Image Processing, Web Scraping, Scientific computing and many more. Python was developed by Guido van Rossum in 1991 and further developed by the Python Software Foundation. Python has a better code readability and simplier syntax as compared to other programming languages like C, C++ or Java. With the help of Python programming language you can work real quick and even more efficiently.
There are two major verision of Python, i.e., Python 2.x and Python 3.x. Programmers now a days a using Python 3.x, so I will also be covering Python 3.x.

## Reason of increased popularity of Python
- Python emphasises on code readability, shorter and easier code and ease of writing.
- Python supports multiple programming paradigms like object-oriented, imperative and functional or procedural programming.
- Python already has inbuilt functions for almost all the job that a programmer needs as a day to day functionality.
- Python provides the user "Ease of coding".
